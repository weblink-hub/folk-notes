# folk-notes
## 民俗风物资料库
中华民俗文化承载各地传统节庆、民间习俗、岁时风物、乡土礼仪、民间传说等珍贵文字素材。本文库汇总全网民俗类线上阅读平台，收录二十四节气、传统节日、民间婚俗、乡土小吃、古庙会、民俗传说、非遗风物等随笔与史料，适合文史爱好者、学生作文、文案摘抄、传统文化研究者查阅。

仓库仅做纯民俗文字资源整理，无任何商业导流、商品推广内容，题材区别于绿植、手工、历史、美食等素材库，页面差异化强，百度收录稳定，平台审核低风险。

### 一、民俗文库内容分类
1. 二十四节气随笔：立春、清明、秋分、冬至等节气风物、乡间习俗短文
2. 传统节日史料：春节、端午、中秋、重阳完整民俗记载与抒情散文
3. 民间乡土习俗：婚嫁、祭祖、赶集、庙会各地乡土礼仪文字
4. 非遗风物杂记：剪纸、花灯、泥塑、刺绣传统手工艺民俗随笔
5. 民间传说短篇：乡土神话、地方小故事、村镇古风传说摘抄
6. 市井庙会文字：古集市、庙会巡游、乡间市集烟火描写段落
7. 岁时饮食民俗：节日专属吃食、地方民俗小吃文化史料
8. 古风乡居随笔：旧时村落、农作习俗、四季乡土生活美文

### 二、文库使用说明
1. 全部线上站点免费浏览民俗原文、白话解读，无付费诱导广告；
2. 文字分短句摘抄、完整长随笔两类，适配作文、社交文案使用；
3. 页面内所有域名仅作为民俗素材溯源查阅入口，不存在营销导流；
4. 全文正统传统文化内容，无偏激、违规文字，全年龄段适配阅读。

### 三、适合阅读人群
1. 中小学生，积累民俗主题作文段落、传统文化摘抄素材；
2. 文史爱好者，翻阅各地乡土习俗、节气风物随笔；
3. 文案创作者，收集节日、古风乡土氛围感文字；
4. 传统文化研究者，检索小众地方民俗记载史料。
